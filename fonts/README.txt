Папка для шрифтов.

Поместите сюда файл font.ttf для использования кастомного шрифта.
Рекомендуемые шрифты:
- Roboto (https://fonts.google.com/specimen/Roboto)
- Open Sans (https://fonts.google.com/specimen/Open+Sans)
- Noto Sans (https://fonts.google.com/noto)

Если font.ttf не найден, система автоматически будет использовать
системный шрифт по умолчанию.
